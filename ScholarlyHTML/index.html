Global Terrorism Visualiser

Autori

Bucătariu Lavinia-Maria și Ghimpu Ioana-Ingrid

Abstract

GLot (Global Terrorism Visualiser) este o aplicație web de vizualizare a atacurilor teroriste din istorie, concepută pentru a fi usor de folosit. Prin intermediul cautarii multi-criteriale, utilizatorii pot explora și analiza datele referitoare la atacurile teroriste din întreaga lume, facilitând înțelegerea și evaluarea impactului acestora asupra comunităților și a societății în ansamblu.

Introducere

Ne propunem să creăm un instrument interactiv și informativ care să ofere utilizatorilor o perspectivă cuprinzătoare asupra fenomenului terorismului global. Dorim să facilităm accesul la date relevante și să permitem vizualizarea acestora într-un mod ușor de înțeles și de explorat. Vrem să contribuim la creșterea conștientizării asupra terorismului global și să ajutăm utilizatorii să înțeleagă mai bine caracteristicile și tendințele acestui fenomen. Prin intermediul statisticilor și vizualizărilor flexibile, dorim să oferim o perspectivă detaliată asupra incidentelor teroriste din întreaga lume și să evidențiem diversele aspecte ale acestui fenomen complex.

Structura

Pentru a realiza interfata web si pentru a implementa functionalitatile aplicatiei folosim:
3.1 HTML

HyperText Markup Language (HTML) este un limbaj de marcare utilizat pentru crearea paginilor web ce pot fi afisate intr-un browser (sau navigator). Scopul HTML este mai degraba prezentarea informatiilor - paragrafe, fonturi, tabele s.a.m.d. - decat descrierea exclusiva a semanticii documentului.

3.2 CSS

CSS (Cascading Style Sheets) este un standard pentru formatarea elementelor unui document HTML. Stilurile se pot atasa elementelor HTML prin intermediul unor fisiere externe sau direct in cadrul documentului html.

3.3 Python

Python este un limbaj de programare versatil, usor de învatat si puternic, folosit in diverse domenii, inclusiv dezvoltarea web, analiza datelor si invatare automata. Cu o sintaxă simpla și expresiva, acesta este utilizat in mod extensiv si pentru dezvoltarea de servere web. Cu Python, dezvoltatorii pot crea rapid si eficient servere web puternice si scalabile.

3.5 MongoDB

MongoDB este o bază de date NoSQL open-source orientată pe documente. Acestă bază de date beneficiază de suport din partea companiei 10gen. MongoDB face parte din familia sistemelor de baze de date NoSQL. Baza de date GLoT va contine cel putin colectiile "attacks" si "statistics".

3.6 Github

GitHub este un serviciu de gazduire web pentru proiecte de dezvoltare a software-ului care utilizeaza sistemul de control al versiunilor Git.

Interfata Aplicatiei

4.1 Home

Pagina de home este alcatuita dintr-un header care contine link-uri catre celelalte pagini si un buton care redirecteaza utilizatorul la pagina de explorare/filtare a atentatelor teroriste din istorie.

4.2 About

Aceasta pagina contine informatii esentiale despre aplicatia web si cum se poate folosi aceasta in beneficiul utilizatorilor. De asemenea va contine o harta cu frecventa atacurilor teroriste in functie de tara. Utilizatorii vor putea experimenta cu aceasta harta pana se vor familiariza cu celalalt mod de cautare si filtrare a diverselor evenimente teroriste.

4.3 Help

Pagina pune la dispozitie o descriere succinta a instrumentului web, care sa il ajute pe utilizator sa foloseasca aplicatia cat mai repede si sa primeasca informatiile de care are nevoie intr-un timp cat mai scurt. Aceasta contine indicatii despre cum se poate folosi filtrarea multi-criteriala si despre cum se pot descarca datele in formatele disponibile pe pagina (CSV, JSON, PNG, SVG). In plus, deoarece vrem sa oferim transparenta utilizatorilor, pe aceasta pagina se vor gasi si sursele de informare folosite de noi (baze de date, stiri).

4.4 Login

Pe pagina de login se vor putea loga doar administratorii acestui site pentru a incarca mai multe date legate de diverse evenimente teroriste (incarcare de fisiere CSV), iar din acest motiv e posibil ca site-ul nostru sa fie updatat periodic cu noi informatii. Utilizatorii nu au nevoie de cont pentru a putea vizualiza si descarca informatiile de pe site, iar functionalitatile vor fi aceleasi pentru oricine ne va accesa site-ul web. Pentru logarea unui admin, acesta are nevoie in prealabil sa ii fie facut un cont de un admin deja existent (initial va fi doar un admin, care ulterior va crea conturile si pentru alti admini daca va fi cazul). Dupa ce are un cont creat si ii sunt asignate drepturile de acces ca si administrator, acesta trebuie sa furnizeze doar emailul si parola.

4.5 Register

Dupa ce un administrator se va loga in contul personal, acesta poate crea noi conturi de admin si garanta drepturi de acces altor persoane. Ulterior si noii admini pot crea alte conturi de admin pentru alte persoane. Nu e de dorit sa fie prea multe conturi de admin pentru a nu compromite integritatea informatiilor de pe site, iar din acest motiv crearea noilor conturi de admini va fi facuta cu atentie.

4.6 Explore

Din aceasta pagina se realizeaza filtrarea multi-criteriala dupa numele tarii, numele orasului si anul atacului terorist. Prin acest tip de filtrare, utilizatorii primesc informatii mai specifice si mai punctuale despre evenimentele cautate.

4.7 Result

Pe pagina cu rezultate se pot vizualiza date in mai multe formate despre atacurile teroriste si de asemenea vor fi disponibile statistici pentru tara, orasul si perioada aleasa. Pe langa vizualizarea informatiilor, utilizatorii pot descarca si rapoarte cu statistici, generate in formatele CSV, JSON, PNG si SVG. Nu exista o limitare la numarul de descarcari pe care le pot realiza utilizatorii.
